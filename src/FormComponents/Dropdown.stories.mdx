### Props

    these are the props that the basic select element takes

    | Prop    | Description                                                                 |  Type                  |
    |-------  |-----------------------------------------------------------------------------|------------------------|
    | label   | this will fill in your label, name, and id for both label and select element| string                 |
    | options | this will populate all your options                                         | string [] or number [] |
    | event   | this will the effect the onChange and onBlur event                          | Function               |
